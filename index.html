Learning Map

In this activity:

  1. You should know
    1. Header .h Files
    2. Implementation .m Files
    3. Functions
    4. Variable Scopes
    5. Debug Logging
  2. We will show you
    1. Custom Classes (Requires: Header .h Files, Implementation .m Files)
    2. Inheritance (Builds on: Custom Classes)
    3. Instance Methods (Builds on: Functions, Variable Scopes)
    4. Class Methods (Builds on: Variable Scopes)
  3. You will research
    1. Singleton Pattern (Requires: Class Methods)
    2. Factory Pattern (Commonly uses: Singleton Pattern)
  4. You will build
    1. Base Class (Based on: Custom Classes)
    2. Subclasses (Requires: Base Class, Inheritance)
    3. Instantiate Objects (From: Subclasses)
    4. Declare Methods (Builds on: Instance Methods)
    5. Override Methods (Requires: Declare Methods)
    6. Call Methods (Uses: Override Methods)
    7. Factory Singleton (Requires: Factory Pattern, Call Methods)
  5. You can go beyond
    1. Categories (Enhances: Declare Methods)
